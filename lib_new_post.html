Logging In

Go to the bottom of the homepage of infra.sarai.net/lib/ and click on Login.

Enter your username and password in the following login page.

Existing Collections

All documents kept in the library portal are part of a collection. Collections are one way of organising the documents stored in the library. The other way of organising documents is by using tags. We will talk about tags later.

Before adding a new post (that is a document) to the library portal, please identify which category it should be part of. A document can only be part of a single category. To see the existing categories, please click on the Collection Tree on the navigation panel at the right side of the screen.

The Collection Tree screen shows the existing collections and their hierarchies (which collection is part of which collection). If the new post can be part of any of the existing collections then proceed to Creating a New Post. If the new post cannot be located within any of the existing collections and you want to create a new collection, then proceed to Creating a New Collection.

Creating a New Collection

To create a new collection, click on the Collections tab on the navigation panel. This takes you to the list of existing collections. Unlike the Collection Tree screen, this does not show the collection hierarchies. Click on Add a Collection to create a new collection.

The portal runs on a software called Omeka, which uses Dublin Core meta-data standards to collect and keep information about the documents stored in the portal. Click here to read more about Dublin Core standards.

The Add a Collection screen has a series of text boxes that can be filled up to enter information about the respective collection. It is important to fill up the following boxes (help in parantheses): Title (name of the collection), Description (a single sentence description of the collection), and Creator (enter 'Sarai-CSDS'). The rest of the boxes can be kept empty.

After filling in the details for the above mentioned boxes, click on Parent Collection tab to go to the screen below and select whether the new collection should be part of any already existing one or not. Finally, choose if the collection should be Public and/or Featured, and then click on Add Collection.

Once created, each collection gets its own page. The entered details can be reviewed and edited from here. Click on the View Public Page to see how the collection appears to visitors to the portal.
